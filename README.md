# 🌱 Bio-Battery Voltage Data Acquisition using LabVIEW & NI-6008

## 📌 Project Overview
This repository contains a LabVIEW graphical programming project designed to interface with an **NI USB-6008 Data Acquisition (DAQ)** device. The primary objective of this system is to accurately measure, monitor, and log the voltage generated by a bio-battery system (Plant Microbial Fuel Cell). This project demonstrates the integration of hardware and software to monitor bio-electricity generation, utilizing components like graphite electrodes and rice plants in a sustainable energy setup.

## ✨ Key Features
* **Hardware Integration:** Seamless communication with the NI USB-6008 DAQ for real-time analog voltage reading.
* **Bio-Electricity Monitoring:** Accurately reads the low-voltage output generated by the bio-battery setup.
* **Interactive Front Panel:** A clear user interface displaying real-time voltage readings and system status.
* **Efficient Block Diagram:** Clean, structured graphical code utilizing NI-DAQmx functions for data acquisition and signal processing.

## ⚙️ Prerequisites
To open, run, or modify the `.vi` files, you will need:
* **NI LabVIEW**
* **NI-DAQmx Drivers** (Essential for the NI-6008 device to communicate with LabVIEW).
* **Hardware:** NI USB-6008 DAQ device & Bio-battery setup.

## 🚀 How to Use
1. Clone this repository: 
   `git clone https://github.com/AhmedKhaled-AgriTech/NI-project.git`
2. Ensure the NI-6008 device is connected to your PC via USB and recognized by NI MAX.
3. Wire the bio-battery outputs (graphite electrodes) to the correct Analog Input (AI) channels on the NI-6008.
4. Open the main `.vi` file using LabVIEW.
5. Run the program from the Front Panel to start acquiring voltage data.
